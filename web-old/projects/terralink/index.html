Project Overview

This is a Minecraft clone I made from scratch in C++. TerraLink generates dynamic landscapes using Perlin Noise. It currently supports singleplayer, multiplayer, terrain manipulation, and player movement physics. Multiplayer consists of a lightweight UDP networking system with TCP fallback for streaming terrain data between clients and dedicated servers.
